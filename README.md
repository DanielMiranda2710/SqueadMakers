# SqueadMakers

# Configuración del proyecto
- Instalar Java JDK 11 
- El driver está en la carpeta "src/main/resources/drivers", si se debe cambiar el driver por el versionamiento es en esa carpeta donde se debe hacer.
- Ejecutar el runner.
